# 스크린샷

메인 [README](../../README.md)의 갤러리가 이 폴더의 PNG를 참조한다. 아래 파일명 그대로
캡처해 넣으면 자동으로 표시된다.

| 파일명 | 담을 화면 |
|---|---|
| `home.png` | 홈 — 씬 + 근무 일정 타임라인 + 🔋컨디션 바 + 📢속보 피드 (히어로) |
| `character.png` | 캐릭터 생성 — 아바타 8종 선택 |
| `market.png` | 시장 리스트 — 국장/미장 종목, 💡팁 배지 |
| `chart.png` | 종목 상세 — 캔들 차트 + 매수/매도 |
| `colleagues.png` | 동료 탭 — 8명 성향·신뢰도·친밀도 |
| `feed.png` | 속보 피드 — 뉴스/단독/지라시 채널 (홈 상단 크롭) |
| `achievements.png` | 업적 시트 — 🏆 달성 목록 |
| `meeting.png` | 회의 미니게임 — 회의 일러스트 + 숫자 찾기 |
| `caught.png` | 미니게임 실패 — "회의 집중 좀 하지?" (고과 -2) |
| `chat.png` | 동료 대화 — 같이 갈지 선택(프롬프트) |
| `chat2.png` | 동료 대화 — 친밀도 + 정보 획득(결과) |
| `dinner.png` | 회식 진입 컷씬 |
| `insider.png` | 내부자 정보 제안 컷씬 — 선택지 포함 |
| `fss.png` | 금감원 조사 컷씬 |
| `promotion.png` | 인사평가 승진 컷씬 |
| `ending.png` | 경제적 자유 엔딩 — 자산 카운트업 |

## 캡처 방법

- 실기기/에뮬레이터, 또는 `flutter run -d chrome`으로 실행 후 캡처. 430×932(포트레이트) 권장.
- 컷씬·엔딩처럼 진행으로 도달하기 힘든 장면은 홈 화면에 임시 디버그 버튼을 달아
  `showCutscene`/`showEnding`을 직접 호출해 찍는 게 빠르다 (촬영 후 제거).
