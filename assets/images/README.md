# 씬 배경 이미지

여기에 이미지 파일을 넣으면 해당 씬 배경으로 자동 사용됩니다. 파일이 없으면
코드가 도형/이모지 플레이스홀더로 폴백하므로 앱은 그대로 동작합니다.

지원(현재):
- `meeting_room.png` — 회의실 씬(회의 미니게임 화면 상단 배경)
- `avatar_<0..7>.png` — 캐릭터 아바타(플레이어·동료 공유 풀)

아바타 번호 ↔ 동료 매핑:
| 번호 | 동료 |
|---|---|
| 0 | 김대리 |
| 1 | 정선임 |
| 2 | 이사원 |
| 3 | 한사원 (인싸) |
| 4 | 박과장 |
| 5 | 최부장 |
| 6 | 윤차장 |
| 7 | 오차장 (투자고수) |

플레이어는 캐릭터 생성에서 고른 번호의 `avatar_<id>.png`를 사용합니다.

넣는 법:
1. 이 폴더(`assets/images/`)에 위 이름으로 PNG를 저장.
2. `flutter run` 재시작(핫 리스타트 `R`) 또는 앱 재실행.
