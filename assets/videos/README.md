# 컷씬 배경 영상

여기에 mp4를 넣으면 해당 컷씬 배경으로 자동 재생됩니다(무음 루프).
파일이 없으면 배경 이모지 Ken Burns 연출로 폴백하므로 앱은 그대로 동작합니다.

지원(현재):
- `scene_meeting.mp4` — 회의 소집
- `scene_smoke.mp4` — 옥상 흡연장
- `scene_coffee.mp4` — 커피 타임
- `scene_lunch.mp4` — 점심시간
- `scene_dinner.mp4` — 회식

넣는 법: 이 폴더에 위 이름으로 저장 후 앱 재실행(핫 리스타트).
세로 화면에 맞게 크롭돼 표시되므로 어느 비율이든 무방합니다.
