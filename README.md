# 주식 인생 (Stock Life Game)

"직장 다니며 주식으로 부자되기" 컨셉의 모바일 싱글플레이 시뮬레이션 게임.
Flutter로 개발하며 Android 우선 출시 후 iOS로 확장한다.

## 게임 컨셉

- 게임 내 하루: 07:00 기상(아침 뉴스) -> 09~18시 근무(고정) -> 저녁 -> 23:30 미장 -> 02:00 취침
- 국장(09:00~15:30)과 미장(23:30~06:00) 두 거래소. 취침 중 미장 잔여 세션은 오버나이트 정산
- 근무 중 몰래 매매: 상사 시선 게이지, 회의 중 몰래 폰 보기 미니게임 (Phase 3)
- GBM 기반 주가 시뮬레이션 + 섹터 상관 + 뉴스 이벤트 점프 (FOMC, CPI, 실적, 횡령 등 22종)
- 목표: 자산 마일스톤 달성, 경제적 자유 엔딩

## 아키텍처

```
lib/
├── engine/        # 순수 Dart 게임 로직 (UI 의존성 0)
│   ├── clock/     #   게임 시간 (하루/틱 상태 머신)
│   ├── market/    #   주가 시뮬레이션 (GBM, 섹터 상관, 이벤트 점프)
│   ├── events/    #   이벤트 테이블·가중치 추첨·효과 관리
│   └── portfolio/ #   매수/매도/손익 정산
├── data/          # 저장/로드 (로컬, Phase 2)
├── ui/            # 화면 (Phase 2)
└── monetization/  # AdMob 래퍼 (Phase 4)
```

엔진은 UI 없이 `flutter test`로 수천 일 몬테카를로 시뮬레이션을 돌려 밸런스를 검증한다.

## 개발

```bash
flutter test      # 엔진 단위 테스트 + 밸런스 리포트
flutter analyze   # 정적 분석
flutter run       # 실행 (Phase 2부터)
```
