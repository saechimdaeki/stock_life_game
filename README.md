# 주식 인생 (Stock Life Game)

"직장 다니며 주식으로 부자되기" 컨셉의 모바일 싱글플레이 시뮬레이션 게임.
Flutter로 개발하며 Android 우선 출시 후 iOS로 확장한다.

## 게임 루프

게임 내 하루: **07:00 기상**(아침 뉴스·월급·귀띔) → **09~18시 근무**(랜덤 일정표) → **저녁**(회식 찬스) → **23:30 미장** → **02:00 취침**(오버나이트 정산).

- 국장(09:00~15:30)과 미장(23:30~06:00) 두 거래소. 미장은 **변동 환율**(일 랜덤워크, 1,250~1,550 밴드)로 원화 환산 매매 — 환차손익 발생
- 1x = 2,500ms/틱(15분). 배속 2x/4x, "다음 개장까지"/"하루 끝" 스킵
- 30일마다 월급, 60일마다 승진(사원 → 임원, 월급 인상)
- Hive 로컬 저장(취침 시점), 구버전 세이브 하위호환

## 구현된 시스템

### 시장

- GBM 기반 주가 시뮬레이션 + 섹터 상관 + 이벤트 점프(호재/악재 ~20종 + 중립 변동성)
- 몬테카를로 1,000일 시뮬레이션으로 밸런스 검증
- 종목 차트: 당일/1주/1달/전체 토글, 새 게임 시 과거 30일 일봉 합성
- **상장폐지**: 바닥까지 붕괴한 종목은 장마감에 폐지 — 보유분은 휴지조각
- **IPO**: 5일차부터 낮은 확률로 대기 풀에서 신규 상장(따끈따끈한 고변동성 종목)

### 직장

- 캐릭터 생성(이름 + 아바타 8종). 아바타는 벡터 기본, `assets/images/avatar_<id>.png` 넣으면 이미지로 교체
- 매일 랜덤 근무 일정표(회의·업무 몰입·보고서·상사 외근·점심). 홈에 하루 종일 표시 + 현재 블록 강조
- 근무 인터랙션(하루가 멈추고 팝업):
  - **회의** → 랜덤 미니게임 5종(눈치 타이밍·연타·블록 부수기·순서 기억·숫자 찾기). 성공 시 [동료와 장난(친밀도)] / [몰래 주식 30초(시간 정지 자유 매매)] 선택
  - **담배타임**(흡연 동료) / **점심**(아무나) / **커피 챗**(인싸) / **회식**(회식러, 저녁) → 같이 가면 친밀도 + 종목 팁
  - 회식은 취함 + 미장 개장까지 시간 점프. 취하면 차트가 춤춘다

### 동료 (8명)

- 성향 7종: 흡연 / 회식러 / 정보통 / 신입 / 일벌레 / 인싸 / 투자고수 — 정보 신뢰도 각자 다름
- 친밀도 0~100 (회의 +4 / 대화 +3 / 회식 +6). 팁 적중률에 가산
- 친밀도 100 보상: 매일 아침 귀띔 + 성향별 보너스(투자고수=재료 잔여일, 인싸=인맥 버프, 일벌레=컨디션 +5)

### 뉴스 피드

- 텔레그램형 속보 피드: 아침 뉴스/공지 + 장중 실시간 속보(채널·시각·톤 색상)
- 속보의 30%는 실제 진행 중인 이벤트 방향을 흘리는 **'단독' 힌트**(방향은 진짜) — 나머지는 노이즈

### 컨디션 (0~100)

- 회식 -25, 심야 매매 -10(하루 1회), 수면 +30(취하면 +20)
- 낮으면: 미니게임 난이도 상승(40 미만), 차트 흐림 + 팁 적중률 하락(30 미만)

### 업적 · 엔딩

- 업적 11종(첫 매매, 자산 마일스톤, 승진, 친밀도, 생존일). 달성 시 알림+피드, 홈의 🏆에서 열람
- **총자산 10억 = 경제적 자유 엔딩** — 엔딩 후에도 계속 플레이 가능

## 아키텍처

```
lib/
├── engine/        # 순수 Dart 게임 로직 (UI 의존성 0)
│   ├── clock/     #   게임 시간·근무 일정표
│   ├── market/    #   주가 시뮬레이션 (GBM, 섹터 상관, 환율)
│   ├── events/    #   이벤트 테이블·가중치 추첨·효과 관리
│   └── portfolio/ #   매수/매도/손익 정산
├── data/          # 세션(월급·승진·친밀도·컨디션·직렬화), 동료, 뉴스 피드, Hive 저장
└── ui/            # 화면·게임 루프 컨트롤러·캐릭터 아바타
```

엔진은 UI 없이 `flutter test`로 수천 일 몬테카를로 시뮬레이션을 돌려 밸런스를 검증한다.

## 개발

```bash
flutter test      # 단위 테스트 + 밸런스 리포트 (52개)
flutter analyze   # 정적 분석
flutter run       # 실행
```

- 디버그 빌드에선 홈에 `🛠 회의⏩ / 🛠 저녁⏩` 점프 버튼이 떠서 인터랙션 테스트가 빠르다
- 아트 에셋은 `assets/images/`에 파일만 넣으면 자동 적용(없으면 코드 폴백) — [assets/images/README.md](assets/images/README.md) 참고
- ⚠️ 미니게임·시트 등 캔버스 UI는 웹 프리뷰 자동화로 검증 불가 — 실기기/에뮬 수동 확인 필요
